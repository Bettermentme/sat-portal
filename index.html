IT OPS – Software Asset Tracker

One-time SharePoint configuration

Step 1 — Azure AD App Registration

Register an app in Azure Active Directory to allow this portal to connect to your SharePoint via Microsoft Graph API.

📋 How to get these values:

1. Go to portal.azure.com → Azure Active Directory

2. App registrations → New registration

3. Add redirect URI:

4. Grant permissions: Sites.ReadWrite.All, User.Read

5. Copy the Client ID and Tenant ID here

Step 2 — SharePoint Site

Enter your SharePoint site URL and the list names where license data will be stored. Lists will be created automatically if they don't exist.

💡 The portal will automatically create SharePoint lists with the correct columns on first run. You can use an existing list if column names match the SOP schema.

Step 3 — Review & Connect

Your configuration will be saved in browser localStorage. Click Connect to authenticate with Microsoft and initialize SharePoint lists.

Configuration is stored locally in your browser only.
